# md_to_json_str.py 说明文档

## 功能概述

`md_to_json_str.py` 是一个**交互式命令行工具**，用于将 Markdown 文本转换为 JSON 字符串格式。  
适用于需要把多行 MD 内容转成可嵌入 JSON 的转义字符串的场景（如写入配置文件、API 请求体等）。

---

## 工作原理

1. 从标准输入逐行读取内容
2. 当输入 `---` 时，将此前收集的所有行合并为一段文本
3. 使用 `json.dumps()` 将文本转成 JSON 字符串（含转义）
4. 将结果输出到标准输出
5. 继续下一轮，可重复粘贴 MD → 输入 `---` → 得到 JSON

---

## 使用方式

### 基本用法

在终端中运行：

```bash
python md_to_json_str.py
```

### 交互流程

1. **启动后**：程序会提示进入交互模式
2. **粘贴 MD**：将 Markdown 内容粘贴到终端（可多行）
3. **输入分隔符**：在新的一行输入 `---` 并按回车
4. **获得输出**：程序会输出对应的 JSON 字符串
5. **继续或退出**：
   - 继续：再次粘贴 MD，输入 `---`，重复
   - 退出：输入 `q`、`exit` 或 `quit` 后回车

### 示例

```
MD → JSON 字符串转换（交互模式）
粘贴 MD 内容，输入 --- 回车 输出 JSON；输入 q 回车 退出
--------------------------------------------------

# 标题
这是一段 **Markdown** 文本。
- 列表项 1
- 列表项 2
---
"# 标题\n这是一段 **Markdown** 文本。\n- 列表项 1\n- 列表项 2"

============================================================

（可继续粘贴下一段 MD...）
q
```

---

## 输入输出说明

| 项目 | 说明 |
|------|------|
| **输入** | 标准输入（stdin），支持多行粘贴 |
| **输出** | 标准输出（stdout），为 JSON 字符串 |
| **提示信息** | 输出到 stderr，不影响管道重定向 |
| **分隔符** | `---`（单独一行）表示「开始转换」 |
| **退出命令** | `q`、`exit`、`quit`（不区分大小写） |

---

## 管道用法

可将文件内容通过管道传入：

```bash
# 在文件末尾追加 --- 后管道传入（需事先编辑）
type your_file.md | echo. & echo --- | python md_to_json_str.py

# 或使用 here-string（PowerShell）
"$(Get-Content your_file.md -Raw)`n---" | python md_to_json_str.py
```

更简单的方式是直接交互粘贴，或配合剪贴板工具使用。

---

## 环境要求

- Python 3.x
- 仅使用标准库（`json`、`sys`），无需安装额外依赖

---

## 典型场景

- 将 Markdown 片段嵌入 JSON 配置
- 在 API 请求中传递多行文本
- 批量将 MD 转为可序列化字符串
- 调试时快速生成 JSON 转义后的文本
